Version 1.4 - October 27th, 2023

Tabs are the focus of this release, as well as several bug-fixes.

Tabs #27
  • Tabs will now appear along the bottom of the screen
  • Swiping left and right in the app will navigate through the open tabs
  • A number of tab-related features can be accessed by long-pressing on a tab
    • Tab icons can be manually set or fetched from the capsule's favicon.txt file if one exists
    • Tabs can be imported or exported as JSON files
    • Tabs can be visually rearranged
  • Tabs can be disabled in the Settings menu
  • Swipe-style tab navigation can also disabled in the Settings menu
Other changes
  • Importing and exporting identities now uses Android's file-browser more seamlessly
  • #24 Gemini links in other apps can now be opened in Rosy Crow
  • #22 Fixed a bug causing empty links to crash the app
Regressions

Due to breaking changes in the MAUI framework, the URL entry can no longer be smoothly animated as it had previously been. This is something that I'll be keeping an eye on in the hopes of fixing in the foreseeable future.
